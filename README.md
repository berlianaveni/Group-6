# Group 6 – K3 PPE Compliance Monitoring System

## 📌 Project Description
This project is developed as part of a collaborative assignment to practice Git and GitHub workflows using Scrum methodology. 

The system focuses on developing an AI-based PPE (Personal Protective Equipment) compliance monitoring solution using computer vision. It aims to detect workers from CCTV footage, identify PPE usage, and provide real-time notifications for safety violations.
